전체동의 항목의 경우 ag_top=true 설정

상세보기 항목의 경우 :
- 기본형과 자세히 보기형, 하위 컨텐츠 추가형으로 나뉜다.
- 기본형 : ag=true 설정
- 자세히 보기형 : ag_more=true 설정
- 하위 컨텐츠 추가형 : exp=true 추가 설정

동의 받는 스크립트는 chk_list 클래스에 체크된 항목을 모두 체크해야만 동의가 완료된다.
