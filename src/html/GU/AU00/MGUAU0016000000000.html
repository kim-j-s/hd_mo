스마트폰
에서
PC
으로 복사하기

스마트폰에 저장된 공동인증서를
PC로 복사할 수 있어요

이용절차

  1. 스마트폰에서 [공동인증서 내보내기] 버튼 선택
  2. 스마트폰에서 PC로 복사할 인증서를 선택하여 인증서 비밀번호 입력
  3. 인증번호 확인 후
    [인증서 내보내기] 버튼 선택
  4. PC의 인증센터 메뉴에서
    [공동인증서 복사(스마트폰 → PC)] 클릭
  5. [공동인증서 가져오기] 선택 하여 새로 열린 팝업창에 스마트폰에서 발급 받은 인증번호 입력
  6. 인증서 저장할 위치 선택 후 [마침] 버튼 선택
{{#> cp/notice dp-active=true }}
  • 스마트폰에 저장된 공동인증서가 없으실 경우 [인증서 가져오기] 메뉴를 통해 PC→스마트폰으로 인증서를 가져 오실 수 있습니다.
  • 공동인증서를 스마트폰으로 가져오기 위해서는 고객님 PC 또는 이동식 디스크 상에 기존에 은행, 증권사 등을 통해 발급받으신 공동인증서가 저장되어 있어야 합니다.
